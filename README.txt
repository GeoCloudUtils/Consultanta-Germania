# Consultanta-Germania (static GitHub Pages)

Folders:
- `/ro/index.html` — Română
- `/ru/index.html` — Русский
- `/de/index.html` — Deutsch
- `/assets/`      — CSS și JS comune

Publicare pe GitHub Pages (Project site):
1. Repo: `Consultanta-Germania`
2. Settings → Pages → Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`
3. Accesează: https://<user>.github.io/Consultanta-Germania/ro/  (și /ru/, /de/)

Editezi textele direct în fișierele limbii respective.
